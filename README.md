# Scored Index

誰にでも作れる程度のものなので, ライセンスはありません.  
見て学んでください.

コマンド実行数は `4 * X + 5` です.  
`X` は `si:args.length - index` ですが, `0` か負の数になってはいけません. (修正予定あり)

## このdatapackが使うもの
* スコアボード  
  * スコアオブジェクティブ
    * `_` ロード時に生成
  * スコアホルダー
    * `index` 引数の値を入れる
    * `#repeat` 内部で用いる
    * `#count` 内部で用いる
* ストレージ
  * `si`
    * `args` 引数の値を入れる
    * `res` 返り値が入る
    * `temp` 内部で用いる

## 使い方
1. `si:args` に配列を, `index` に番号を入れる
2. `function scored_index:index` を叩く
3. `si:res` には `si:args[index]` が入る
